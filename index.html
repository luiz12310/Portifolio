HTML5

CSS3

CSS3 é uma linguagem de estilo usada para descrever a apresentação dos elementos da página os estilizando.

JavaScript

JavaScript é uma linguagem de programação usada no desenvolvimento web para funcionalidades e comportamentos da página.

Angular

AngularJS é um framework usado para construir aplicativos para web baseados em uma única página dinâmica.

C#

SQL

SQL é uma linguagem padrão para manipulação de registros em bancos de dados relacionais.
